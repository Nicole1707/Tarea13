Principales Compiladores utilizdos en la actualidad

1. GCC (GNU Compiler Collection)
Convierte el código fuente en una secuencia de tokens, identificando palabras clave, operadores y símbolos.
2. Análisis Sintáctico
Verifica la estructura gramatical de los tokens, generando un árbol de sintaxis si el código es correcto.
3. Análisis Semántico
Comprueba el significado de las estructuras sintácticas, validando tipos de datos y referencias.
4. Optimización
Mejora el código intermedio para reducir el tiempo de ejecución y el uso de recursos sin cambiar su funcionalidad.
5. Generación de Código
Convierte el código optimizado en instrucciones de lenguaje de máquina para la arquitectura de destino.
6. Optimización de Código
Realiza ajustes finales al código de máquina para mejorar la eficiencia y el rendimiento. Se traduce a un lenguaje de bajo nivel o lenguaje máquina, específico para la arquitectura de la computadora en la que se ejecutará.
7. Linking
Combina el código objeto con otros módulos necesarios (como bibliotecas externas) y resuelve las referencias entre ellos para producir el ejecutable final.
